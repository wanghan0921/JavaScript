在es6之前, 对象不是基于类创建的, 而是利用构造函数来定义对象和它的特征

## 1. 创建对象的三种方法

```js
// 利用 new Object()
var obj1 = new Object()


// 理由字面量创建
var obj2 = {}


// 利用构造函数
function Person(name, age) {
  this.name = name
  this,age = age
  this.skill = function() {
    console.log('111')
  }
}
var obj3 = new Person('wanghan', 26)
```
