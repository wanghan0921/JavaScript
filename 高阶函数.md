### 高阶函数

 定义 : 高阶函数是对其他函数进行操作的函数, 它接收函数作为参数或将函数作为返回值输出
 
 ```js
 function fn(callback) {
    callback && callback()
 }
 fn(function() {...})
 ```

```js
function fn() {
  return function() {}
}
```

这样的函数都成为 高阶函数


#### 闭包

1. 什么是闭包 
  
  闭包是一个函数(一个作用域可以访问另外一个函数的局部变量)
  
2. 应用 

  延展局部变量的生命周期 , 防止全局变量名污染

3. 缺点

 可能会有内存泄漏
